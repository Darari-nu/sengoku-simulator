# 川中島の戦い 3D実装要件

対象ページ: `battles/kawanakajima/index.html`

## 実装方針

川中島は第四次川中島合戦を基礎に、「妻女山」「八幡原」「海津城」「千曲川」の位置関係と、武田別働隊・上杉本隊のすれ違いを見せる。啄木鳥戦法と霧中遭遇は伝承を含むため、本実装では「戦術理解のための可視化」として扱う。

## 時刻別要件

| 時刻 | 要件 | スクショ確認 |
|---|---|---|
| 5:00 | 妻女山と八幡原が朝霧に包まれる。ただし部隊ラベルは読める | 濃霧でも上杉・武田の位置が分かる |
| 6:10 | 武田別働隊が海津城方面から妻女山へ進む | 黄色い別働隊が西北方向へ進む |
| 7:20 | 上杉軍が妻女山を下り、八幡原へ進む | 青い上杉軍が中央へ移動している |
| 8:00 | 霧が薄れ、上杉本隊が武田本陣前に現れる | 中央で青と黄の部隊が接近 |
| 8:40 | 上杉軍が武田本陣へ攻勢をかける | 上杉謙信隊が武田信玄方面へ迫る |
| 9:30 | 武田別働隊が八幡原へ戻り、挟撃構図になる | 別働隊が中央へ戻る |
| 10:00 | 上杉軍が千曲川方面へ退く | 青い部隊が北東方面へ退く |

## 霧要件

- 5:00から7:30は霧を強くするが、部隊名・旗・字幕を完全に隠さない
- 8:00以降は八幡原中央が見える程度まで霧を下げる
- 検証モードでは霧をさらに薄くして動線確認を優先する
