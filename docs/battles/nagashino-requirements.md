# 長篠の戦い 3D実装要件

対象ページ: `battles/nagashino/index.html`

## 実装方針

長篠は「馬防柵」「鉄砲隊」「武田騎馬隊の突撃失敗」を見せる。史実には兵力・三段撃ちの解釈に幅があるため、本実装では設楽原の地形、連吾川、柵、突撃と射撃の関係を理解する可視化として扱う。

## 時刻別要件

## 演出要件

- 馬防柵は地形名ラベルと部隊の間に視覚的な線として読ませる
- 長篠城は設楽原の東側、鳶ヶ巣山方面と同じ武田側背後に置く
- 鉄砲射撃は矢印と字幕で表現し、部隊を隠す煙は使わない
- 検証モードでは部隊ラベル、柵、突撃方向が読めること
