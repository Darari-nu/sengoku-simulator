# 合戦地形・布陣監査 2026-07-01

目的: 既存18合戦について、川・山・城・橋などの地形軸に対して部隊の初期位置と移動方向が破綻していないか確認する。これは合戦理解のための相対配置監査であり、精密な史跡座標再現ではない。

## 判定

| 合戦 | 地形上の基準 | 実装確認 | 判定 |
|---|---|---|---|
| 関ヶ原 | 笹尾山・松尾山・南宮山と中央戦場 | 東軍が東側から中央、西軍が山側・西側から受け、小早川が松尾山方面から大谷隊へ下る | OK |
| 桶狭間 | 善照寺砦、田楽狭間、狭い谷筋 | 織田軍が砦側から田楽狭間へ進み、今川本陣を突く | OK |
| 川中島 | 妻女山、八幡原、海津城、千曲川 | 上杉軍が妻女山から八幡原へ下り、武田別働隊が妻女山へ向かう構図 | OK |
| 長篠 | 設楽原、連吾川、馬防柵、長篠城 | 初回監査後に再確認。馬防柵を連吾川西岸寄りへ、長篠城を設楽原東側へ修正し、武田軍が東から連吾川を越えて柵前で止まる構図にした | 修正済 |
| 山崎 | 天王山、山崎中央、勝竜寺城方面 | 羽柴軍が西から圧迫し、明智軍が東側へ後退 | OK |
| 三方ヶ原 | 三方ヶ原台地、街道、浜松城 | 徳川軍が浜松城方面から出て、武田軍の圧迫で浜松城方向へ退く | OK |
| 小牧・長久手 | 小牧山、長久手、別働隊の南下路 | 徳川・織田方が小牧山で固定し、羽柴別働隊が長久手方面で捕捉される | OK |
| 厳島 | 海峡、宮尾城、塔岡、島内の狭地 | 毛利軍が海上から上陸し、陶軍本営と退路を圧迫 | OK |
| 姉川 | 姉川、野村、三田村、小谷方面 | 修正前は川と平行に横衝突していた。修正後は織田・徳川を南岸、浅井・朝倉を北岸に置き、渡河後に野村・三田村方面で衝突 | 修正済 |
| 河越夜戦 | 河越城、包囲陣、入間川筋 | 北条救援隊が城外から包囲陣へ突入し、守備隊が城内から呼応 | OK |
| 賤ヶ岳 | 賤ヶ岳、余呉湖、北国街道 | 羽柴軍が街道・山道を押し上げ、柴田方突出部が北東へ崩れる | OK |
| 本能寺の変 | 本能寺境内、京の町筋、包囲線 | 明智軍が複数方向から本能寺を包囲し、織田方は寺内で孤立 | OK |
| 天王寺・岡山 | 天王寺口、岡山口、徳川本陣、大坂城方面 | 豊臣方が城外から徳川本陣へ突撃し、徳川方が側面から包囲 | OK |
| 耳川 | 耳川、高城方面、大友前線、島津陣 | 大友軍が川へ進出・渡河し、島津軍が東側から反撃して押し戻す | OK |
| 人取橋 | 人取橋、本宮方面、連合軍前線 | 伊達軍が橋頭西側で受け、連合軍が東から圧迫。退却方向も本宮方面へ戻る | OK |
| 月山富田城 | 月山富田城、菅谷口、補給路、包囲線 | 毛利軍が山城を囲み、小早川隊が補給路を遮断 | OK |
| 大坂夏の陣 | 大坂城、天王寺口、岡山口、徳川本陣 | 豊臣方が城外へ出て徳川本陣へ突撃し、徳川方が突出部を包囲。終盤字幕は実動に合わせて修正 | 修正済 |
| 大坂冬の陣 | 大坂城、外堀、真田丸、徳川砲列 | 豊臣方が城・真田丸を保持し、徳川方が外側から突撃・砲撃 | OK |

## 修正内容

- `anegawa` の地形軸を再整理した。
- 姉川は東西に流れる川として残し、織田・徳川軍を南岸、浅井・朝倉軍を北岸へ配置した。
- 野村・三田村のラベルを北岸側へ寄せ、渡河と衝突の説明も南岸から北岸への動きに変更した。
