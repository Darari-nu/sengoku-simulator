# 三方ヶ原の戦い 3D実装要件

対象ページ: `battles/mikatagahara/index.html`

## 実装方針

三方ヶ原は「武田軍の圧迫」「徳川軍の野戦出撃」「徳川軍敗走と浜松城への帰還」を見せる。布陣の詳細には解釈差があるため、本実装では台地、街道、浜松城の位置関係と、徳川軍が押し返される流れを理解する可視化として扱う。

## 時刻別要件

| 時刻 | 要件 | スクショ確認 |
|---|---|---|
| 14:00 | 武田軍が三方ヶ原台地へ進む。徳川軍は浜松城方面から出る | 台地、街道、浜松城、両軍初期位置が読める |
| 15:00 | 徳川軍が野戦に出て武田軍と接触する | 家康・本多隊が中央へ前進 |
| 16:00 | 武田先鋒と山県隊が徳川軍を圧迫する | 武田軍が徳川軍正面と側面に近づく |
| 17:00 | 徳川軍が崩れ始める | 徳川本隊が浜松城方向へ向きを変える |
| 18:00 | 家康が浜松城へ敗走し、武田軍が台地を制する | 徳川側が南東へ退き、武田側が中央を押さえる |

## 演出要件

- 徳川軍は一度中央へ出た後、浜松城方向へ戻るパスにする
- 武田軍は複数方向から圧力をかける構図にする
- 夕刻表現は暗くしすぎず、敗走方向が読めるようにする
