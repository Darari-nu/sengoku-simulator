# 山崎の戦い 3D実装要件

対象ページ: `battles/yamazaki/index.html`

## 実装方針

山崎は「中国大返し後の秀吉接近」「天王山周辺の位置取り」「短時間での明智軍崩壊」を見せる。天王山占拠の具体は解釈に幅があるため、本実装では地形と戦線の圧迫を理解する可視化として扱う。

## 時刻別要件

| 時刻 | 要件 | スクショ確認 |
|---|---|---|
| 15:00 | 両軍が山崎周辺に布陣。天王山と淀川が読める | 羽柴軍が西、明智軍が東にいる |
| 16:00 | 羽柴軍が天王山周辺へ圧力をかける | 池田・天王山周辺隊が山側へ近づく |
| 17:00 | 中央で羽柴軍が攻勢を強める | 秀吉・中川隊が明智本隊へ接近 |
| 18:00 | 明智右翼が崩れ、光秀本隊が後退する | 青い明智軍が東へ下がり始める |
| 19:00 | 明智軍が勝竜寺城方面へ敗走する | 明智側表示が薄くなり、東へ退いている |

## 演出要件

- 天王山は高い地形として見せ、戦場の要としてラベルを置く
- 短期決戦なので時間軸は午後から夕刻までに絞る
- 夜演出は濃くしすぎず、終盤も部隊ラベルを読めるようにする
